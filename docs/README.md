# Mask Studio Complete Pack

This bundle merges the strongest parts of the rebuild phases into one standalone Flutter package.

Included:
- image-space masking editor
- layer-aware mask workflow
- selection + proposal workflow
- LaMa submit/status/result pipeline
- binary mask export
- validation + refinement services
- local segmentation bridge
- SAM-ready remote engine scaffold
- advanced preview widgets/painters

Notes:
- This is the most complete standalone pack generated in the session.
- The remote SAM bridge still needs a matching backend endpoint.
- Because the original project shell was only partially available, this bundle is delivered as a standalone feature app rather than a guaranteed drop-in replacement for every original file.
